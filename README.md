# Gemini RP 專案

歡迎來到 Gemini RP！這是一個實驗性的角色扮演遊戲專案，由 Gemini CLI 驅動，讓大型語言模型（LLM）作為您的遊戲管理員（GM）。我們採用了受 Obsidian Vault 和 SillyTavern LoreBook 啟發的系統，旨在提供一個沉浸式、動態且易於擴展的文字冒險體驗。

## 專案特色

*   **LLM 驅動的 GM：** Gemini CLI 將根據您的輸入、預設規則和動態情境來敘述故事、扮演 NPC 並管理遊戲世界。
*   **知識庫管理 (Vault)：** 遊戲的所有資訊（角色、地點、物品、任務等）都儲存為結構化的 Markdown (`.md`) 檔案，易於閱讀和修改。
*   **動態情境注入 (LoreBook)：** 遊戲會根據您的對話內容，智慧地從知識庫中提取相關資訊，確保 GM 的回應始終貼近上下文。
*   **結構化初始化 (Session Zero)：** 遊戲開始時，GM 將引導您進行「零號會議」，共同設定遊戲的類型、基調、玩法風格和內容偏好。
*   **清晰的溝通機制：** 透過特定的語法，您可以在遊戲中與 GM 進行「遊戲外 (OOC)」溝通。
*   **一致的敘事風格：** 遊戲將採用統一的第三人稱過去式敘事，提供更沉浸的閱讀體驗。
*   **動態劇情推進 (Fronts)：** 世界中的危機和情勢會自行發展，創造一個不斷演進且充滿活力的故事。
*   **純敘事判定 (Narrative-First Resolution):** 遊戲不依賴骰子或數值，而是根據角色的能力描述、情境風險和遊戲基調來共同創造故事。

## 如何開始遊玩

### 1. 前置準備

*   **安裝 Gemini CLI：** 確保您的系統已安裝並配置好 [Gemini CLI](https://github.com/google/generative-ai-docs/tree/main/gemini-cli) 工具。
*   **克隆專案：** 如果您尚未克隆，請將此專案克隆到您的本地機器上：
    ```bash
    git clone [專案的 Git 連結]
    cd GeminiRP
    ```
*   **初始設定：** 本專案已經自動為您建立了必要的資料夾結構 (`Campaign/`, `Characters/`, `Locations/`, `Items/`, `Quests/`) 和核心規則檔案 (`GEMINI.md`)。

### 2. 啟動遊戲

在專案的根目錄下，啟動 Gemini CLI 並開始您的冒險：

```bash
# 根據您的 Gemini CLI 啟動指令
gemini-cli # 或您設定的其他指令
```

### 3. 「零號會議 (Session Zero)」引導

*   一旦啟動，Gemini CLI 將會以 GM 的身份，引導您進行「零號會議」。這是一個互動式的訪談過程，GM 會詢問您一系列問題來設定遊戲的基調、類型、玩法風格和內容邊界。
*   您的回答將被記錄在 `Campaign/Session_Zero.md` 檔案中，作為遊戲的核心設定。

### 4. 進行遊戲

*   **自然語言互動：** 以您的角色身份，透過自然語言輸入您的動作、對話或思考。
*   **遊戲外溝通 (OOC)：** 如果您需要對遊戲規則、劇情走向提出建議，或者單純想跟 GM 聊天，請使用方括號 `[...]` 包裹您的訊息。
    *   **範例：** `[OOC: 我覺得這裡是不是可以安排一個突發事件？]`
*   **GM 的回應：** GM 將根據您的輸入、知識庫中的資訊以及設定的規則來生成劇情描述、NPC 對話和世界回應。GM 的敘述將統一採用第三人稱過去式。

### 5. 管理遊戲世界 (知識庫 Vault)

您可以隨時查看或修改 `Campaign/`、`Characters/`、`Locations/`、`Items/`、`Quests/` 等資料夾中的 `.md` 檔案。
*   每個 `.md` 檔案都代表一個遊戲實體。
*   檔案頂部的 `keys:` 元數據欄位定義了觸發該資訊的關鍵字。您可以自行添加或修改這些關鍵字來擴展 GM 的知識。

## 專案結構

除了上述提到的知識庫資料夾外，專案還包含以下兩個輔助資料夾：

*   `Templates/`: 包含所有知識庫檔案的格式範本和最佳實踐範例。當您想創建新的角色、地點或其他實體時，可以參考這些範本來確保格式的一致性。
*   `Tests/`: 包含用於驗證遊戲系統穩定性的測試案例。`Tests/Stress_Test.md` 檔案描述了一系列的極端場景，可用於手動測試 GM 的行為是否符合預期。

享受您的冒險吧！如果您有任何建議或遇到問題，請隨時提出。
